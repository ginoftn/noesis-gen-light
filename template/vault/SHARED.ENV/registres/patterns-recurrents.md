# Patterns recurrents

*Patterns detectes a travers les sessions, consolides regulierement.*
*Un pattern passe en "Actif" apres 3+ occurrences sur 2 semaines.*
*Un pattern est "Resolu" quand il n'a pas ete observe pendant 3+ semaines.*

*Exemples de patterns courants a surveiller :*

- **Proliferation des debuts** -- Nouveau projet > projet en cours. Le debut est toujours plus excitant que le milieu.
- **Theorisation de l'evitement** -- "Ca mature" au lieu de "j'evite". Discours sophistique qui protege de l'action.
- **Contenant avant contenu** -- Construire l'outil parfait avant de faire le travail. L'organisation comme procrastination elegante.
- **Burst sans suite** -- Production massive puis silence. Le momentum se dissipe avant de devenir une habitude.

## Notes

- Detecter un pattern n'est pas le juger. C'est le rendre visible pour que le choix devienne conscient.
- Toujours verifier le contexte complet avant de diagnostiquer un pattern.
- Les patterns ne sont pas des defauts -- certains sont des methodes de travail deguisees.
